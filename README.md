Employee Management System (SQL)

A simple SQL-based relational database project for managing employees, departments, customers, and orders. The project demonstrates database design and common SQL operations, including data retrieval, filtering, sorting, aggregation, grouping, and subqueries.

Features
Employee and department management
Customer and order records
Aggregate functions and grouping
Sorting and filtering queries
Subqueries for analytical operations
Technologies Used
SQL
Relational Database Management System (RDBMS)
